# SKS
This repository provides code for the paper "Hate Speech Detection based on Sentiment Knowledge Sharing"

![avatar](figure1.jpg)

# Requirements
Python>=3.7

tensorflow>=2.0

# Data
[DV][1], [SE][2], [SA][3]

# Usage
After download the data and the pre-trained word vectors, just run the sample_run.sh
